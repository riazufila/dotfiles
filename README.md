# dotfiles

The root directory of this repository acts as the home directory of your Linuxdistribution.

For example, .vimrc and .Xresources should be placed in your home directory such as "/home/riazufila/" or "~" to be simple. The same logic applies to folders. For example, .config should be put in home directory such that it can be presented as "/home/riazufila/.config", and any dotfiles inside should be placed as is.
